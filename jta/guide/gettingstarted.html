Getting started

Download

To get the API, you need a Maven project. You can see the maven repository and the dependency here:

<repository>
  <id>jitpack.io</id>
  <url>https://jitpack.io</url>
</repository>

<dependency>
  <groupId>com.github.NitramMann21</url>
  <artifactId>JTA</id>
  <version>VERSION</id>
</dependency>

The library has been coded in Java 11. In consequence, you should use Java 11 or newer.

Creating your first bot

Creating a bot is actually surprisingly simple*. Create an application in the Twitch Developer Console, which requires 2FA enabled. After initializing the API and calling JTABotBuilder.create("YOURCLIENTID", "YOURCLIENTSECRET"); you already have got a bot that can do almost everything the library offers.

public static void main(String[] args) {
  // Initialize the library.
  JTA.initialize();

  // Create a bot.
  JTABotBuilder.create(clientId, clientSecret);
}




*See Ceave Gaming
