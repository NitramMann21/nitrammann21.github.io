User and Channel Management

You may want to manage some stuff, e.g. a channel's stream title. Just call Channel#setStreamTitle(String) and...

Nope, not gonna work. You will get an ErrorResponseException. What does this mean and how can you fix this? The reason is simple: You don't have the permission to manage the stream.
Now, how can you fix this? You need to set the user access token. Next question: How do you get user access tokens? You can follow this guide on the JTA's GitHub repository.

Please note that you can only manage the user who was logged in when authorizing.

Here is a list of some methods which need user access tokens:

  • Channel#setStreamTitle(String)
  • Video#delete()
  • User#getBlockedUsers()
  • Channel#startPrediction(String, String, String, int)
