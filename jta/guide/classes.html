Classes

JTA

The main class of the library: You can not create instances. This is an utility class which is used by the whole API. It saves data like the HTTP client or the logger.

JTABot

This class represents a bot which can send messages, manage stream data and much more! To get an instance, call JTABotBuilder#create(String, String).

User

This class represents a twitch user. You can get blocked users, the follows, the stream schedule and much more! To get an instance, call JTABot.getUserByName(String) or JTABot.getUserById(long).

Channel

This class represents a twitch channel. You can get banned users, the chat, the follower count, the team, the user, and much, MUCH more! Also, you can set the stream language / title, start a poll and a prediction. To get an instance, you need an user. Then just call getChannel() and you have the channel.
